üö¶ Getting Started

# Clone this repository of wonders
git clone https://github.com/yourusername/textadventureservices.git

# Enter the dungeon
cd textadventureservices

# Summon the dependencies
go mod download

# Set up your OpenAI API key
# Create a .env file with your OpenAI API key:
echo OPENAI_API_KEY=your_api_key_here > .env

# Enable PowerShell script execution (run PowerShell as Administrator)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# Start the game (this will open separate windows for each service)
.\start_game.ps1

That's it! The script will:
1. Start all services in separate color-coded windows
2. Wait for services to initialize
3. Open a command console with ready-to-use game commands

Note: If you get a security warning about running the script, you can use:
powershell -ExecutionPolicy Bypass -File .\start_game.ps1

üéÆ Sample Commands

# Look around the room

# Check your inventory
curl -X POST http://localhost:8080/api/v1/process-input -d '{"input": "inventory"}'

# Take an item
curl -X POST http://localhost:8080/api/v1/process-input -d '{"input": "take keyboard"}'

# Move around
curl -X POST http://localhost:8080/api/v1/process-input -d '{"input": "go north"}'

# Check game state
curl -X GET http://localhost:8080/api/v1/game-state
